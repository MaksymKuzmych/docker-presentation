DOCKER

Created by Maksym Kuzmych

Why you should use Docker:

  • Isolation
  • Run on different devices
  • All dependencies inside

Docker components:

  • Container
  • Image
  • Repository
  • Registry

Docker commands:


              docker ps
            

              docker ps -a
            

              docker images
            

              docker run hello-world
            

              docker run -it busybox

Useful flags:

  • -d
  • -it
  • -p 8080:80
  • -v

              docker run -v ${PWD}:/usr/share/nginx/html

Images


              FROM node:alpine

              WORKDIR /app

              COPY . .

              RUN npm install

              CMD ["node", "index.mjs"]

docker build -t my-app .


              docker run -p 5000:5000 -t my-app

Docker compose

Docker compose benefits:

  • Declarative approach
  • Single Command Usage
  • Auto creating images
  • Auto creating network
  • Interaction using service names

              version: '3'

              services:
                app:
                  build: ./app
                mongo:
                  image: mongo
